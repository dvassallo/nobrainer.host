nobrainer.host

From http://localhost:3000 to online in seconds

Folder-based Routing

Organize your apps in folders. Each folder becomes a subdomain.

/myapp → https://myapp.yourdomain.com
/blog → https://blog.yourdomain.com
/api → https://api.yourdomain.com

Braindead deployments

Configure your settings once.

npx nobrainer.host setup

Deploy directly to the server.

npx nobrainer.host deploy

Automate deployments with GitHub Actions.

npx nobrainer.host automate

Static & Docker

Mix and match static sites and Docker apps. Just drop an index.html or docker-compose.yml in a folder.

Auto SSL

Every subdomain gets a Let's Encrypt certificate automatically. Renewals are handled for you.

Own it

Deploy to your own VPS. No platform fees, no limits. You control the infrastructure.

Zero Config

No complex YAML files or routing rules. The file structure is the configuration.
